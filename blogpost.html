prueba del uso de la aplicacion git

en esta prueba utilizaremos los comandos de git bash y el uso de git como lo son "cd" para ir a la raiz de mi pc "pwd" para ver en que lugar exacto del ordenador estoy, git ini para iniciar la parte de git, git add "nombre del archivo" para agregarlo a standind y git commit para agregarlo al archivo final y actualizarlo, con el git commit se recomienda usar git commit -m "comentario para saber que cambios hubo en esa version" , ahora probaremos lo que se pueden hacer con las ramas, lo increible que es el mundo de las ramas que son el git branch.
